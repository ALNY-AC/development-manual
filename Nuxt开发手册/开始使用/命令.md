# 命令与部署



## 命令

````json
//package.json
{
  "scripts": {
  "dev": "nuxt",
  "build": "nuxt build",
  "start": "nuxt start",
  "generate": "nuxt generate"
}
````

执行方式：`npm run xxx`



### 安装项目依赖

````
npm i
````



### 启动项目

```
npm run dev
```



### 编译项目

```
npm run build
```



### 启动服务端渲染服务器

```
npm run start
```



### 编译静态项目

```
npm run generate
```
